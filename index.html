ภาพรวมผลงาน (Overview)

ในปี 2568 ได้ออกแบบและพัฒนา 4 โปรเจกต์ที่เชื่อมโยงกันอย่างมีระบบ โดยมุ่งเน้นการ วิเคราะห์เชิงกลยุทธ์ และ การสื่อสาร Insight อย่างมืออาชีพ ทั้งในรูปแบบ Dashboard, Website และ HTML Presentation แบบ Interactive

โปรเจกต์สำคัญ (Projects)

1. Alert CFC Dashboard

กลยุทธ์การบริหารความเสี่ยงด้านคลังสินค้า

  • วิเคราะห์ DOI, Turnover, การจัดกลุ่มพื้นที่ผิดปกติ และ Risk Alert
  • ใช้สูตร DAX แบบ All-in-One เพื่อวิเคราะห์และสร้างการแจ้งเตือนอัตโนมัติ
  • รองรับการวิเคราะห์รายพื้นที่/เขต สื่อสารประเด็นสำคัญให้ผู้มีส่วนเกี่ยวข้อง

2. StockView Dashboard

วิเคราะห์ประเภทสินค้าและทุนจม

  • วิเคราะห์ประเภทสินค้า (Fast/Slow/Non), ABC, DOI, Aging, Inventory Value
  • ใช้สูตร DAX รวมมิติเพื่อ Insight ที่ต่อเนื่องทั้งกราฟและตาราง
  • ช่วยให้หน่วยงานมองเห็นโครงสร้างคลังสินค้าและวางแผนเชิงกลยุทธ์ได้ชัดเจน

3. Power BI Learning Website

พัฒนาองค์ความรู้เชิงกลยุทธ์ในองค์กร

  • เว็บไซต์เพื่อสอน Power BI สำหรับการวิเคราะห์เชิงกลยุทธ์
  • รวมความรู้ด้าน Data Model, DAX, Power Query, UX Design
  • มีบทเรียน + ตัวอย่าง Dashboard และแนวทางวิเคราะห์ที่ตอบโจทย์กลยุทธ์
  • ใช้ GitHub Pages เผยแพร่ รองรับทั้ง Desktop & Mobile
  • ✳️ ให้คำปรึกษาและสอนแบบตัวต่อตัวสำหรับผู้สนใจภายในองค์กร

4. HTML Strategic Insight Presentation

เปลี่ยน Dashboard ให้เป็น Web Presentation สุดล้ำ

  • สรุป Insight จาก Alert CFC และ StockView เป็น Web Dashboard แบบ Interactive
  • ใช้ Reveal.js สำหรับ Presentation Slide และ Chart.js สำหรับกราฟ
  • HTML + Tailwind CSS เพื่อความสวยงามระดับมืออาชีพ
  • ใช้ JSON เป็น data source ทำให้สามารถเปลี่ยนเดือนและดู Insight ได้แบบ Dynamic
  • เผยแพร่ผ่าน GitHub Pages ให้ผู้ใช้งานเข้าถึงได้ตลอดเวลา

บทบาทและความรับผิดชอบ (Role & Contribution)

  • Power BI Developer: Data Modeling, DAX, Power Query
  • Insight Architect: สกัด Insight สำคัญเพื่อใช้ในการสื่อสารอย่างเป็นระบบ
  • Web Developer: HTML/CSS/JS + Chart.js + Reveal.js + Tailwind
  • UX/UI Designer: ออกแบบการนำเสนอให้เหมาะกับกลุ่มเป้าหมาย
  • Trainer: ถ่ายทอดความรู้ในรูปแบบ Website และการสอนแบบ Coaching

  • ผู้เรียน One-on-One หลายคนสามารถสร้าง Dashboard จริงได้หลังเรียน
